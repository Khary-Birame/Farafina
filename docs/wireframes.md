## Composants Réutilisables

### Header
- Logo Farafina
- Navigation principale
- Bouton CTA
- Menu mobile (hamburger)

### Footer
- Liens rapides
- Informations de contact
- Réseaux sociaux
- Newsletter
- Mentions légales

### Cards
- Card Programme
- Card Statistique (KPI)
- Card Témoignage
- Card Partenaire

## Responsive Design

### Mobile (< 768px)
- Navigation hamburger
- Stack vertical pour les cards
- Grille 1 colonne

### Tablet (768px - 1024px)
- Navigation adaptée
- Grille 2 colonnes
- Cards redimensionnées

### Desktop (> 1024px)
- Navigation complète
- Grille 3-4 colonnes
- Layout optimisé

## Notes de Design

- Utilisation de la charte graphique vert/gris anthracite
- Espacement cohérent (8px grid system)
- Typographie hiérarchisée
- Contraste suffisant pour l'accessibilité
- Animations subtiles et professionnelles
